# Alessandro De Luca - Developer Portfolio 💻

A cyberpunk/retro-terminal inspired personal portfolio showcasing projects, skills, and background. It features a custom interactive terminal easter egg, dynamic ASCII background graphics, and smooth interactions.

## 🌟 Features

- **Modern Cyberpunk Aesthetic**: High-contrast dark mode with neon accents and sleek UI typography.
- **Dynamic ASCII Orb Background**: A persistent, animated, GSAP-powered ASCII orb and film grain that seamlessly persists across all sections.
- **Hidden "Hacker Terminal" Easter Egg**: A fully interactive, hidden command-line interface. Includes features like simulated system hacking (`hack`), matrix effects (`matrix`), neofetch (`neofetch`), and standard commands (`whoami`, `skills`, `projects`). Access it via the "Terminal" nav link or by typing `hack` anywhere on the page!
- **Internationalization (i18n)**: Built-in bilingual support (Italian/English) for seamless localized experiences.
- **Responsive Design**: Fluidly adapts to mobile, tablet, and desktop viewports with a polished mobile navigation menu.

## 🛠 Tech Stack

- **Frontend Core**: Vanilla HTML5, CSS3 (CSS Variables, Flexbox/Grid), and modern JavaScript (ES6+).
- **Animations**: [GSAP](https://gsap.com/) & ScrollTrigger for high-performance scroll animations and canvas-based graphics.
- **Build Tooling**: [Vite](https://vitejs.dev/) for an ultra-fast local development server and optimized production builds.

## 🚀 Local Setup

To run this project locally, ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. Clone the repository and navigate into the project directory.
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Open the provided `localhost` URL in your browser.

## 📦 Build for Production

To create an optimized production build:

```bash
npm run build
```

This will generate a `dist` folder containing the minified and optimized assets ready for deployment.
